01 Technical SEO — 60/100

Critical Content-Security-Policy absente T1
Header Content-Security-Policy non renvoye. Risque XSS/clickjacking et signal de qualite degrade.
Definir une CSP adaptee aux tiers integres (GTM, Avis Verifies, etc.).
Critical X-Frame-Options absente T2
Risque de clickjacking. Le header n'est pas renvoye.
Ajouter X-Frame-Options: SAMEORIGIN.
Warning Referrer-Policy absente T3
Fuite potentielle de referrer vers les domaines tiers.
Ajouter Referrer-Policy: strict-origin-when-cross-origin.
Warning Permissions-Policy absente T4
Acces aux API navigateur (camera, micro, geoloc) non restreint.
Ajouter Permissions-Policy: camera=(), microphone=(), geolocation=().
Warning Redirection racine en 302 au lieu de 301 T5
abracadaroom.com → 302 → /en/ (1455 ms). Link equity non transmise. De plus, un visiteur FR est redirige vers /en/ au lieu de /fr/.
Passer en 301. Idealement, detecter Accept-Language et rediriger vers /fr/ par defaut pour les francophones, avec un x-default en fallback.
Warning Pas de Sitemap dans robots.txt T6
La decouverte du sitemap depend uniquement de GSC.
Ajouter Sitemap: https://www.abracadaroom.com/sitemap.xml dans robots.txt.
Info AI crawlers non geres dans robots.txt T7
11 user-agents IA (GPTBot, ClaudeBot, PerplexityBot, Google-Extended, CCBot...) sans politique explicite. Pour un site qui veut apparaitre dans les reponses IA, il est recommande de les autoriser explicitement.
✓ HTTPS + HSTS (max-age=31536000; includeSubDomains)
✓ X-Content-Type-Options: nosniff

02 On-Page SEO — 70/100

Warning H2 mal structures (texte decoartif fusionne) O4
Les H2 concatenent le texte CSS ::before/::after avec le titre. Ex : "The icing on the cake1 booking = 1 good deed". Bots et lecteurs d'ecran confus.
Separer le label decoratif (CSS) du texte semantique. Le H2 ne doit contenir que le titre reel.
Warning Aucun H3 sur la page O5
Hierarchie plate, sections non scannables par les bots et les utilisateurs.
Ajouter des H3 pour les cards de destinations et categories d'hebergements.
Warning Hreflang sans x-default O6
5 langues declarees (en/de/fr/es/nl) mais aucun fallback pour les langues hors perimetre.
Ajouter <link rel="alternate" hreflang="x-default" href="https://www.abracadaroom.com/">.
Info Hreflang /de/wishlist/ non exclue dans robots.txt O7
Les wishlist sont exclues en es, en, fr, nl mais pas en de. Verifier si /de/ est pleinement deployee.
✓ Title 54 car. (bien calibre)
✓ Meta description 157 car.
✓ Canonical self-referentielle correcte
✓ H1 unique et pertinent

05 Image Optimization — 50/100

Critical Images avec src="" (elements casses) I1
Logo et badge Avis Verifies ont un src vide. Elements casses pour les user-agents sans JS, LCP degrade.
Servir le src en HTML natif (utiliser data-src pour le lazy-load JS si besoin, mais fournir un fallback).
Critical Aucune image avec width/height I2
25 images parsees, toutes sans dimensions. CLS degrade (Core Web Vital direct).
Ajouter width et height explicites sur chaque <img>.
Critical Aucune image avec loading="lazy" I3
25/25 images sans loading specifie. LCP et bande passante impactes.
loading="lazy" sur les images sous le fold, fetchpriority="high" sur l'image LCP.
Warning Alt text logo avec typo "Abracadroom" I4
Le logo footer a un alt "Abracadroom" au lieu de "AbracadaRoom". Mot-cle brand manque.
Corriger en "AbracadaRoom".
Warning Formats .jpg/.png uniquement I5
Poids 2 a 5x superieur vs AVIF/WebP.
Servir en AVIF/WebP via <picture>.
